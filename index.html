Publications

My current research focuses on the Distributed, Optimized control of a dynamic system composed of multiple agents, each with a linear behavior. We are currently implementing a decentralized, multi-agent reinforcement learning approach to address this problem optimally.

Distributed, Optimized control of a stochastic Non-consensus Opinion dynamic network

Bachelor's thesis

Recently, we have updated the environment to a scale-free, linear, social network with stochastic, stubborn behavior. What makes this environment different is the fact that it will never reach a consensus on its own. Our Multi-agent Reinforcement-Learning approach will try to force the system to reach a favorable state of consensus. However, other decentralized programming algorithms are being studied for a better and more generalized answer. In this system, because of the large size of the environment and multiple input edges of the dynamic system, a decentralized controlling system is vital.

Additional details will be shared in the future. Codes, results, and details of this research will be posted fully in the corresponding repository after the submition. However, If you have any questions, please feel free to contact me via email for more information.

Autonomous Unmanned Aerial Vehicle Indoor Navigation Based on Deep Learning Approaches.

International Conference on Robotics and Mechatronics. (ICROM)

In this research two deep learning approaches for autonomous drone navigation were introduced. The first method uses a convolutional neural network to identify safe paths, while the second employs the drone’s camera and generative networks to create depth images. Both methods save costs and energy by relying on an RGB camera while the GAN-based method provides faster decisions.
